# Esmerald Tower Web 💎

Proyecto de sitio web premium para la exclusiva **Esmerald Tower**.

## 🚀 Cómo desplegar en GitHub Pages

Para alojar esta página en GitHub de forma gratuita, sigue estos pasos:

1.  **Crea un repositorio en GitHub**: Ponle de nombre `EsmeraldTowerWeb`.
2.  **Sube los archivos**:
    - `index.html`
    - `style.css`
    - `script.js`
    - `hero-tower.png`
3.  **Activa GitHub Pages**:
    - Ve a la pestaña **Settings** (Configuración) de tu repositorio.
    - En el menú lateral izquierdo, haz clic en **Pages**.
    - En **Build and deployment** > **Branch**, selecciona `main` (o `master`) y la carpeta `/ (root)`.
    - Haz clic en **Save**.
4.  **¡Listo!**: En unos minutos, tu página estará disponible en `https://tu-usuario.github.io/EsmeraldTowerWeb/`.
